## 验收标准

- [ ] 完整用户流程跑通：
  1. 输入有效债券基金代码（020741），查询成功，展示基金信息预览
  2. 点击确认，跳转结果页，自动触发计算
  3. 加载完成后展示完整 8 项指标
  4. 点击「刷新计算」可重新计算
- [ ] 异常状态前端展示验证：
  - [ ] 网络错误 → 展示"网络异常，请检查网络连接" + 重试按钮
  - [ ] 基金不存在（错误码 40001）→ 展示"未找到该基金（代码：XXXXXX），请检查后重新输入"
  - [ ] 非债券型基金（错误码 40002）→ 展示"该基金为[类型]，当前仅支持债券型基金"
  - [ ] 部分指标缺失（null）→ 标注"N/A"，不阻塞其他指标展示
- [ ] 加载状态验证：
  - [ ] 骨架屏在加载中正常展示
  - [ ] 按钮加载态（spinner + 禁用）正常
- [ ] 移动端基础适配：
  - [ ] iPhone SE（375px 宽）下页面无横向溢出
  - [ ] 卡片最大宽度约束正常

- [ ] 免责声明在所有页面可见（FundInput + FundResult）
