## TC-4.3-001: 完整用户流程 — 输入债券基金代码并完成计算
- **类型**: 功能测试
- **关联验收标准**: 完整用户流程验证
- **前置条件**: 
  - 后端服务运行于 localhost:8000
  - 前端开发服务器运行于 localhost:5173
  - MongoDB Atlas 连接正常
  - akshare 数据源可用
- **输入**:
  - 基金代码：`020741`
- **执行步骤**:
  1. 打开 FundInput 页面
  2. 在搜索框输入 `020741`
  3. 点击「查询」按钮
  4. 等待 FundPreview 组件展示基金信息（名称、类型、净值）
  5. 点击「确认并查看详情」按钮
  6. 页面跳转至 FundResult
  7. 等待计算结果展示
- **预期输出**:
  - FundPreview 展示基金名称为"国泰君安 30 天滚动持有债券"
  - 路由跳转至 `/result` 页面
  - ResultHeader 展示基金名称和代码
  - YieldMetrics 展示 8 项收益指标（至少部分指标有值）
  - MarketRates 展示 10 年期国债收益率和信用利差
  - DisclaimerBar 在页面底部可见
- **清理**: 无
